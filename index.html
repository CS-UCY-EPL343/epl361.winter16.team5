Πληροφορίες

Πιο κάτω αναγράφονται πληροφορίες για τις δύο υπηρεσίες.

  • Γραμμή Καταγγελιών (Hotline)


    -Σχετικά με την Γραμμή Καταγγελιών

    Κύριος στόχος της Γραμμής Καταγγελιών είναι να συμβάλει ενεργά στην μάχη εναντίον της παρανομίας στο διαδίκτυο και παρέχοντας έτσι ένα ασφαλέστερο διαδίκτυο για μικρούς και μεγάλους. Ακόμα εξασφαλίζεται μία συνεργασία με διεθνείς φορείς και γραμμές καταγγελιών για συντονισμό όλων των χωρών στην προσπάθεια καταπολέμησης του φαινομένου αυτού.

    Η υπηρεσία Hotline δίνει στο χρήστη τη δυνότητα να κάνει μία ανώνυμη (εάν επιθυμεί) καταγγελία, για θέματα που αναγράφονται πιο κάτω, και η οποία αφού εξεταστεί από κάποιο λειτουργό μας, θα διαγραφεί, αφού πρώτα σταλεί ανώνυμα στην αστυνομία.

    Είδη καταγγελίας Hotline
    • Παιδική Πορνογραφία
    • Ρατσισμός/ Ξενοφοβία
    • Κλοπή Προσωπικών Δεδομένων
    • Παραβίαση Απορρήτου Επικοινωνιών
    • Διαδικτυακός Εκφοβισμός
    • Σεξουαλική Παρενόχληση
    • Ηλεκτρονική Αποπλάνιση
    • Εμπορικοί Κίνδυνοι/Απειλές

  • Γραμμή Βοήθειας (Helpline)


    Σχετικά με την Γραμμή Βοήθειας

    Η Γραμμή Βοήθειας Helpline έχει ως στόχο να παρέχει στον κόσμο τα μέσα για να μιλήσουν ανώνυμα, ή μη, σε εκπαιδευμένους επαγγελματίες που μπορούν να τους στηρίξουν, σε ότι ανησυχία ή απειλή μπορεί να αντιμετωπίσουν μέσω του διαδικτύου και να τους συμβουλέψουν, είτε ψυχολογικά, είτε τεχνικά, για αντιμετώπιση του προβλήματός τους. Προσφέρει την ευκαιρία στο κοινό να θέσει ερωτήματα και να εκφράσει τις ανησυχίες του ή τις απορίες του σχετικά με επιβλαβείς διαδικτυακές συμπεριφορές, επιβλαβή επικοινωνία και επιβλαβές περιεχόμενο στο διαδίκτυο. Η Γραμμή παρέχει ψυχολογική και τεχνική υποστήριξη.

    Η Γραμμή Βοήθειας χειρίζεται θέματα όπως:
    • Παιδική Πορνογραφία
    • Ρατσισμός/ Ξενοφοβία
    • Κλοπή Προσωπικών Δεδομένων
    • Παραβίαση Απορρήτου Επικοινωνιών
    • Διαδικτυακός Εκφοβισμός
    • Σεξουαλική Παρενόχληση
    • Εθισμός Στο Διαδίκτυο
    • Ηλεκτρονική Αποπλάνιση
    • Ανεπιθύμητη Αλληλογραφία
    • Εμπορικοί Κίνδυνοι/Απειλές

Προσωπικά Στοιχεία (Προαιρεστικά)
Στοιχεία καταγγελίας

*: υποχρεωτικό πεδίο

Αυτή η καταγγελία, αφού εξεταστεί, θα σταλεί ανώνυμα στην αστυνομία για εσάς. Εάν δεν επιθυμήτε κάτι τέτοιο και σκοπός σας είναι να συνομιλήσετε με κάποιον λειτουργό τότε παρακαλούμε όπως αποταθείτε στην υπηρεσία Helpline.

Όλες οι πληφορορίες και προσωπικά στοιχεία είναι και θα παραμείνουν εμπιστευτικά. Οι καταγγελίες διαγράφονται μετά την αποστολή τους στην αστυνομία εκτός εάν εσείς επιθυμείτε διαφορετικά.

Σας προτρέπουμε όπως μας δώσετε προσωπικά στοιχεία εάν κρίνεται πως είναι αναγκαία για την καταγγελία σας.

Στις λεπτομέρειες μπορείτε να δώσετε περισσότερες πληροφορίες σχετικά με την καταγγελία σας καθώς και προσωπικά στοιχεία δικά σας ή και κάποιου άλλου προσώπου σε περίπτωση που κρίνετε πως είναι αναγκαία για την καταγγελία ή και επιθυμείτε να λάβετε πληροφορίες σχετικά με την εξέλιξή της.

Καταγγελία Helpline


Προσωπικά Στοιχεία
Στοιχεία καταγγελίας

*: υποχρεωτικό πεδίο

Στις λεπτομέρειες μπορείτε να δώσετε περισσότερα προσωπικά στοιχεία που θα χρησημοποιηθούν στην επικοινωνία μαζί σας. Προτρέπουμε όπως δώσεται ακόμα περισσότερες πληοροφορίες που μπορεί να χρειάζονται σχετικά με την καταγγελία.

Το πεδίο email είναι υποχρωτικό, αφού θα χρησιμοποιηθεί από τους λειτουργούς μας για επικοινωνία μαζί σας.

Υπενθυμίζουμε πως η καταγγελία Helpline θα καταλήξει σε κάποιο λειτουργό μας, ο οποίος θα επικοινωνίσει μαζί σας μέσω email, για συζήτηση του προβλήματος. Εάν επιθυμείται η καταγγελία σας να καταλήξει ανώνυμα στην αστυνομία παρακαλούμε όπως αποταθείτε στην υπηρεσία Hotline.

Οποιαδήποτε προσωπικά στοιχεία σας είναι και θα παραμείνουν εμπιστευτικά.
